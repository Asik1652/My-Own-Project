Compass Mobile Application Documentation

Overview

Compass is a comprehensive mobile application designed for educational institutions, providing different interfaces and functionalities for four distinct user personas. The application is built using React Native, ensuring cross-platform compatibility for both iOS and Android devices.

Technology Stack

  • Frontend: React Native
  • State Management: Redux
  • Navigation: React Navigation
  • API Integration: Axios
  • Styling: Custom Theme System

User Personas and Features

1. Student Persona

Accessible Pages:

Home Dashboard

  • Personalized view of academic information
  • Quick access to important features
  • Notifications and announcements

TimeTable

  • View daily and weekly class schedules
  • Period-wise subject allocation
  • Teacher information for each subject

Attendance

  • View personal attendance records
  • Attendance percentage calculation
  • Subject-wise attendance breakdown

Leave Request

  • Submit leave applications
  • Track leave request status
  • View leave history

Answer Sheets

  • Access to exam answer sheets
  • View teacher comments and feedback
  • Download capability for offline access

Marksheet

  • View examination results
  • Subject-wise mark breakdown
  • Performance analytics and progress tracking

2. Teacher Persona

Accessible Pages:

Home Dashboard

  • Class overview and schedules
  • Pending tasks and notifications
  • Quick access to teaching tools

Attendance Entry

  • Mark student attendance
  • View class attendance statistics
  • Generate attendance reports

Mark Entry

  • Input student examination marks
  • Grade assignments and tests
  • Performance analysis tools

Homework Upload

  • Assign homework and assignments
  • Set submission deadlines
  • Track student submissions

Leave Approval

  • Review student leave requests
  • Approve/reject applications
  • Leave history management

3. Administrator Persona

Accessible Pages:

Home Dashboard

  • Institution-wide analytics
  • System notifications
  • Quick access to administrative tools

Admission Management

  • Process new admissions
  • Student database management
  • Document verification system

Staff Management

  • Teacher allocation
  • Staff attendance tracking
  • Performance monitoring

Support Ticket System

  • Handle technical issues
  • Track request status
  • Resolution management

4. Parent Persona

Accessible Pages:

Home Dashboard

  • Child's academic overview
  • Important notifications
  • Fee payment status

Academic Progress

  • View attendance records
  • Access mark sheets
  • Track homework completion

Communication

  • Teacher communication channel
  • View school announcements
  • Parent-teacher meeting scheduling

Conclusion

The Compass mobile application provides a comprehensive solution for educational institutions, effectively serving the needs of students, teachers, administrators, and parents through a well-organized, feature-rich platform. The application's modular architecture ensures scalability and maintainability, while its intuitive interface provides an excellent user experience across all personas.
